# M24AID022_CSL7110_Assignment2
Assignment2 : Min-Hashing and LSH Course: CSL7110 – Machine Learning with Big Data

Environment: Windows 11 Frameworks: Google Collab

Overview : This repository contains the code , files, datasets used for Min-Hashing and LSH Assignment2 for CSL7110.

The assignment has 5 questions

Questions 1–3 are based on given minhash.zip file which has 4 files D1, D2, D3, D4

Questions 4–5 are based on older MovieLens 100k dataset (5MB), which consists of a set of 943 users who have rated 1682 movies. (Download link: https://files.grouplens.org/datasets/movielens/ml-100.zip)
